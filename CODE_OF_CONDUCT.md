# Code of Conduct

Language: [中文](CODE_OF_CONDUCT.zh-CN.md)

This project uses a direct, issue-first collaboration style. Be specific, respectful, and evidence-driven.

## Expected Behavior

- Keep discussion focused on Guard, Handoff, Audit, Shield, and the public matrix proof.
- Report bugs with reproducible steps and synthetic examples.
- Respect privacy boundaries. Do not post private source, raw diffs, prompts, model responses, secrets, credentials, or customer data.
- Accept maintainer decisions on scope boundaries for prerelease work.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing security details before maintainers have a chance to respond.
- Repeatedly pushing out-of-scope product claims such as certification, hosted verification, public leaderboard status, or OS sandbox guarantees without an accepted release decision.

## Enforcement

Maintainers may edit, hide, lock, or close discussions that violate this code. Serious or repeated violations may result in blocking from the project spaces.
