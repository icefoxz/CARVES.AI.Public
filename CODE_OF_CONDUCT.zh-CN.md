# 行为准则

语言：[英文](CODE_OF_CONDUCT.md)

本项目采用直接、issue-first 的协作方式。请具体、尊重他人，并以证据为基础讨论。

## 期望行为

- 让讨论聚焦于 Guard、Handoff、Audit、Shield 和公开 matrix proof。
- 使用可复现步骤和 synthetic examples 报告 bug。
- 尊重隐私边界。不要发布私有源码、raw diff、prompt、模型回复、secret、credential 或客户数据。
- 对 prerelease work 的 scope boundary，接受 maintainer 的决定。

## 不可接受行为

- 骚扰、威胁、人身攻击或歧视性语言。
- 在 maintainer 有机会回应前公开安全细节。
- 在没有 accepted release decision 的情况下，反复推动 certification、hosted verification、public leaderboard status 或 OS sandbox guarantees 等越界产品 claim。

## 执行

Maintainer 可以编辑、隐藏、锁定或关闭违反本准则的讨论。严重或重复违规可能导致被项目空间屏蔽。
